# Basic Networking Commands

Linux provides several tools to test and troubleshoot networks.

---

## Check IP Address

$ ip addr

---

## Check Network Interface

$ ip link

---

## Test Connectivity

$ ping google.com

---

## Routing Information

$ route -n  
$ ip route

---

## DNS Lookup

$ nslookup google.com  
$ dig google.com

---

## Trace Network Path

$ traceroute google.com

---

## Host Configuration

Edit hosts file:

$ sudo nano /etc/hosts
