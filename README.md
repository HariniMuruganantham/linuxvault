
## About This Repository

This repository contains structured Linux notes converted from handwritten classroom material into clean Markdown files.

It is designed for beginners and intermediate learners who want a clear, practical reference for Linux system administration.

## How to Use

1. Begin with `basics/linux-introduction.md`  
2. Continue through folders in order:  
   `shell` → `filesystem` → `users-groups` → `permissions`  
3. Use the `cheatsheets` folder for quick revision and practice.
